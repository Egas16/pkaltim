# Assets Folder

Folder untuk menyimpan aset-aset project seperti gambar, dokumen, dan screenshot.

## Struktur Folder

```
assets/
├── screenshots/          # Screenshot website dari setiap tim
│   ├── tim-1.png
│   ├── tim-2.png
│   └── ...
├── 12-tim-assignment.png # List pembagian 12 kelompok
├── navies-subtema.png    # Screenshot subtema dari Navies
└── repo-screenshot.png   # Screenshot repo GitHub
```

## Usage
- `screenshots/` - Untuk submission screenshot dari tim
- Root assets - Untuk dokumentasi project umum
