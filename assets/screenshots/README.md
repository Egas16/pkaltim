# Screenshots Folder

Folder ini untuk menyimpan screenshot website dari setiap tim.

## Format Penamaan
- `tim-1.png` - Screenshot Tim 1
- `tim-2.png` - Screenshot Tim 2
- dst.

## Requirements
- **Format:** PNG atau JPG
- **Ukuran:** Max 2MB
- **Resolusi:** Min 1920x1080 (Full HD)
- **Content:** Homepage atau landing page utama

## Cara Upload
1. Ambil screenshot full page website kalian
2. Rename file: `tim-[nomor].png`
3. Copy ke folder ini
4. Commit dan push ke GitHub

## Notes
- Screenshot harus tampilan desktop (bukan mobile view)
- Pastikan UI terlihat jelas dan professional
- Tidak ada watermark atau overlay
